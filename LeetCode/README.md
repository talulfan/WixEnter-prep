# LeetCode Practice (JavaScript)

Structured like the main prep workspace: **practice first**, then compare with **solutions**.

## Folder layout

```
LeetCode/
├── README.md
├── exercises/     # Problem statement + empty function + tests
└── solutions/     # Full solution + complexity + INTERVIEW TIP
```

## How to work in VS Code

1. Open `WixEnter-prep` (or this `LeetCode` folder) in VS Code.
2. Implement in `exercises/<file>.js`.
3. Run: `node exercises/course-schedule.js`
4. After your attempt, read `solutions/<same-file>.js`.
